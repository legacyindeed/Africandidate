# AfriCandidate

Your MBA Journey, Simplified.

AfriCandidate is a comprehensive resource platform designed specifically for African students pursuing MBA programs in the United States. From application to graduation and beyond, we provide the tools and information you need to succeed.

## Features

- **Top 30 MBA Schools** - Rankings, deadlines, tuition, and international student stats
- **Pre-Arrival Guide** - Everything you need to prepare before landing in the US
- **Budget & Funding** - Cost of living calculator and scholarship resources
- **Banking & Mobile** - Best banks and phone carriers for international students
- **Recruiting** - Career resources and recruiting timelines
- **Community** - Connect with fellow African MBA students
- **Health Insurance** - Navigate the US healthcare system

## Tech Stack

- React
- Vite
- React Router
- Formspree (contact form)
- Vercel (hosting)

## Getting Started

```bash
npm install
npm run dev
```
